# java-web-gradle-blade-hello-world

## Description
A POC for blade framework rendering html page.

## Tech stack
- cors

## Docker stack
- gradle:7.6.4-jdk11-alpine

## To run
`sudo ./install.sh -u`
Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
